# 🎓 Hamdel School Management System

A modern, responsive School Management System developed with **Python, Flask, SQLite, Bootstrap 5, and Google Cloud**. The application helps schools efficiently manage students, teachers, classes, attendance, and academic records through a simple and user-friendly web interface.

---

## 🚀 Features

### ✅ Dashboard
- Professional admin dashboard
- Student statistics
- Teacher statistics
- Class statistics
- Attendance overview

### 👨‍🎓 Student Management
- Add new students
- Edit student information
- Delete students
- Search students
- Responsive student table

### 👩‍🏫 Teacher Management
- Add new teachers
- Edit teacher information
- Delete teachers
- Search teachers

### 📚 Upcoming Features
- Class Management
- Attendance Tracking
- Grade Management
- Student Reports
- Authentication & Login
- Role-Based Access
- PDF Report Export
- Charts & Analytics

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/ghafoorhamdel/Hamdel-School-Management-System.git
```

Go to the project folder:

```bash
cd Hamdel-School-Management-System
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser:

```
http://localhost:8080
```
